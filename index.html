Oque é HTML?

(Voltar)

O HTML é um sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto —, foi criado pelo Físico britânico Tim Berners-Lee em 1991. Foi criado com a intenção de corrigir um problema de comunicação entre ele e seus colegas, Berners-Lee queria uma forma simples de compartilhar suas pesquisas e resultados com o pessoal. De mãos dadas com a internet, naquela época, o HTML foi altamente reconhecido por sua simplicidade.

E claro, vale ressaltar que não se trata de uma Linguagem de programação, afinal para que seja uma liguagem de programação é necessário que seja possível aplicar uma lógica, e também possa criar funcionalidades dinâmicas.

Como funciona o html

(voltar)

O HTML, ou HyperText Markup Language, funciona como uma linguagem de marcação que define a estrutura e o conteúdo de uma página web. Ele utiliza tags para envolver diferentes partes do conteúdo, indicando ao navegador como exibir essas partes. As tags são organizadas hierarquicamente, formando uma estrutura de árvore que representa a relação entre os elementos na página. Além disso, as tags podem ter atributos para fornecer informações adicionais.

Em resumo, o HTML atua como um esqueleto que define a estrutura básica e o conteúdo de uma página web, permitindo que os navegadores interpretem e exibam o conteúdo de maneira adequada.


Falando sobre tags

(voltar)

As tags HTML são códigos especiais usados para marcar e estruturar o conteúdo em uma página da web. Elas são delimitadas por colchetes angulares ("<>" ) e fornecem instruções para o navegador sobre como exibir diferentes partes do conteúdo. Em essência, as tags HTML atuam como rótulos que dizem ao navegador o que fazer com o texto ou elementos dentro delas.

Por exemplo, a tag "p" é usada para indicar o início de um parágrafo, enquanto "/p" indica o seu fim. As tags desempenham um papel crucial na criação e formatação de páginas web, permitindo a apresentação estruturada e compreensível do conteúdo.


Estrutura básica do HTML

(Voltar)

O documento HTML sempre inicia com o que chamamos de estrutura básica. Esta estrutura é quase que imutável. Sempre será dessa forma e você sempre começará seu HTML por esse código. É possível compreender o documento em HTML de uma maneira muito simples, através de uma divisão de blocos das tags essenciais, conforme a a seguinte estrutura:

  • Definição do documento (doctype)
  • Cabeça (head)
  • Corpo (body)

    • Ficaria assim:

        <html>
    <head>
        <title>Título da página</title>
    </head>
    <body>
        Aqui vai o código HTML que fará seu site aparecer.
    </body>
    </html>

    Textos

    (Voltar)

  • Elementos de Cabeçalho (H1, H2, H3, etc.):
    • Os elementos de cabeçalho em HTML, representados pelos tags < h1> a < h6>, são usados para definir títulos e sub-títulos em uma página. Eles seguem uma hierarquia, onde < h1> é o mais importante (texto maior) e < h6> é o menos importante (texto menor).
  • Parágrafos:
    • O elemento < p> é utilizado para definir parágrafos em HTML. Ele cria uma quebra de linha antes e depois do conteúdo do parágrafo, tornando o texto mais organizado e legível.


    Listas ordenadas e não ordenadas

    (Voltar)

    Em HTML, podemos criar listas ordenadas e não ordenadas.

  • Listas não ordenadas (< ul >):
    • As listas não ordenadas são usadas para representar itens que não têm uma ordem específica. Os itens são geralmente marcados com marcadores, como pontos ou círculos.
  • Listas ordenadas (< ol >):
    • As listas ordenadas são usadas quando a ordem dos itens é importante. Os itens são numerados automaticamente.

    Links

    (Voltar)

    Em HTML, os links são utilizados para criar referências a outros recursos na web, como páginas da web, documentos, imagens, vídeos, entre outros. Eles são implementados por meio do elemento (anchor ou âncora).

  • Estrutura básica de um link: A estrutura básica de um link em HTML é a seguinte:
    • < a href="URL_do_destino">Texto do Link < /a >

    Sendo, < href >: Atributo que especifica o endereço do destino para onde o link aponta. Pode ser uma URL absoluta (começando com "http://" ou "https://") ou uma URL relativa (referenciando um recurso na mesma página ou diretório). E "Texto do Link"será exibido na página e clicável para acessar o destino.

  • Exemplo de link:
    • < p>Clique < a href="https://www.dio.me">aqui< /a> para visitar o site de exemplo.< /p>

    Neste exemplo, "aqui" é o texto do link, e ao clicar nele, o usuário será redirecionado para o site "https://www.dio.me".


  • Alvo do Link (Atributo target):

    • O atributo target é usado para especificar onde o novo documento será aberto. Os valores comuns incluem _blank (abrir em uma nova janela ou guia) e _self (abrir na mesma janela). Exemplo:

    < a href="https://www.dio.me" target="_blank"> Dio.me < /a>

  • Links Internos (Âncoras):

    • Para criar links internos em uma página (âncoras), você pode usar o atributo href com o ID do elemento de destino. Aqui, o usuário é redirecionado para o tópico desejado. Ao clicar no link, a página rolará até a seção identificada pelo ID.

    Os links são essenciais para a navegação na web, permitindo que os usuários acessem diferentes recursos e páginas com facilidade. Eles desempenham um papel crucial na construção de uma estrutura de navegação eficaz em páginas HTML.
